Listing of all available courses.
Course Overview

This is a dynamically generated page based on the URL parameters sent.

Each course will have their own landing page where students can sign up for a class; or if a student is already enrolled, they can link to the first course module.

If security is on, then "signup for class" will only show if user hasn't enrolled in class. Similarly, link for "go to first module" will only show if security is on and student is enrolled.
Course Module

This is a dynamically generated page based on the URL parameters sent.

Each course will have one or more modules that will minimally be in one language.
About

This is an "about" page that describes the mission of "New Tech Learning"
Create Account

This allows creation of user student account. Once security is in place, the link to this page will only show if the user is not logged in.
Login

This allows the user to login. Once security is in place, the link to this page will show only if the user is not logged in

Logout

This allows the user to logout. Once security is in place, the link to this page will show only if the user is already logged in.
